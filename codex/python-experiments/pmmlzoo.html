Generating data ¶

Data is generated for three main types of inputs:

  • Continuous numerical values

  • Discrete numerical values

  • Categorical values

The process is similar for all three, where an histogram is interpolated from the points supplied, and consequently data is sample from that histogram.

Let’s look at some examples.

Continuous data ¶

Note

create_numerical_series uses cubic spline interpolation, which means you have to provide at least four data points.

Let’s start with creating an histogram which will represent the distribution of an income variable. You can supply the “seed” data in the form of a pair (income, frequency) . The frequency is a relative measure, so don’t worry about the actual value.

For instance, just for illustration, if we wanted an income distribution where 45 year olds earned double than the 20 and 60 year old we could either do: 

import pmmlzoo.pmmlzoo as zoo

age_examples = [[20.0, 1.0], [45.0, 2.0], [60.0, 1.0]]
series = zoo.create_continuous_histogram(start=20, end=60, data=age_examples, size=100)

import matplotlib.pyplot as plt

plt.scatter(series[:,0], series[:,1], s=2)
plt.xlabel("Age")
plt.ylabel("Frequency")
plt.title("Age distribution")
plt.show()

We can now sample from this histogram: 

data = zoo.sample_from_histogram(series, size=1000)
