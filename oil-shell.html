Installation

macOS

Linux

The instructions to install on Linux are available in the Oil INSTALL document. Basically, download the release and run a make installation:

$ wget # latest release
$ tar -x --xz < oil-$VERSION.tar.xz
$ cd oil-$VERSION
$ ./configure
$ make
$ sudo ./install
